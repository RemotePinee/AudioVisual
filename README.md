## 安装

在本地安装并运行此项目，你需要先安装 [Node.js](https://nodejs.org/)。

1.  克隆代码仓库：
    ```bash
    git clone https://github.com/RemotePinee/AudioVisual.git
    ```
2.  进入项目目录：
    ```bash
    cd audiovisual
    ```
3.  安装依赖：
    ```bash
    npm install
    ```

## 使用

要启动应用程序，请运行以下命令：

```bash
npm start
```
