# AudioVisual 项目归档说明

## 项目状态

本仓库已归档，原项目源码、构建产物、依赖文件和历史提交记录已从公开远程仓库中移除。

当前仓库仅保留此说明文档，用于说明项目状态、保留原仓库入口，以及保留既有的 Star、关注记录和相关仓库元信息。

## 为什么清理仓库

AudioVisual 曾是一个个人学习和实验性质的桌面应用项目。随着项目阶段结束，原有代码已经不再继续维护，也不再适合作为公开仓库继续分发。

因此，本仓库已经完成清理：

- 不再提供源码下载；
- 不再提供构建脚本；
- 不再提供安装包或发布说明；
- 不再接受功能请求；
- 不再处理新 Issue 或 Pull Request；
- 不再维护旧版本兼容性。

## 关于历史代码

公开远程仓库的默认分支历史已被重写。重写后的仓库只保留当前文档，不再包含原项目文件或旧提交记录。

如果你曾经 fork、clone 或下载过旧版本代码，请自行判断是否继续保留。本仓库当前不会再提供旧代码、旧版本包或相关技术支持。

## 关于 Star

删除仓库会导致 Star 一并消失。为了保留曾经关注、收藏和支持过本项目的记录，本仓库选择保留仓库本体，但移除项目代码。

感谢所有曾经点过 Star、提出反馈或关注过这个项目的朋友。这个仓库现在更像是一块项目墓碑：代码不再公开，记录仍然保留。

## 关于后续维护

本项目后续没有公开维护计划。

如将来重新启动类似方向的项目，可能会使用新的仓库、新的代码结构和新的项目名称，而不会继续沿用当前仓库的旧代码历史。

## 仓库状态摘要

- 项目名称：AudioVisual

- 源码状态：已从公开远程仓库移除
- 历史提交：已重写
- 维护状态：停止维护
- 文档用途：说明归档原因并保留仓库入口

## 致谢

感谢所有曾经关注、使用、收藏或反馈过 AudioVisual 的朋友。

项目到这里告一段落，仓库保留下来，是为了给过去留一个清楚、干净的结尾。
